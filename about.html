About

I write about technology, software engineering, and the craft of building things for the web. Occasionally with interactive visualizations.

Hi — I'm a software engineer interested in the intersection of design and technology. I believe the best technical writing doesn't just explain concepts, it makes them tangible through clear prose and, where it helps, interactive demonstrations.

This blog is a space for me to explore ideas, share things I've learned, and build small experiments. Some posts are traditional long-form articles, while others are more visual and interactive — incorporating embedded visualizations, live code examples, and data explorations.

I'm particularly interested in web architecture, developer tooling, data visualization, and the design of systems that are both powerful and humane.

About this site

This site is built with plain HTML, CSS, and JavaScript — no frameworks, no build step. It's intentionally simple: fast to load, easy to maintain, and focused on the content. The design is inspired by the clean, utilitarian aesthetic of modern developer-focused publications.
